NAME
       skinedit - a skin editor for TiEmu and VTi (TI's hand-helds emulators)


SYNOPSYS
       skinedit [-h] [-v] [--help] [--version] skin_to_open


DESCRIPTION
       The  skinedit program is a skin editor. It allows you to load/save/con-
       vert VTi and TiEmu formatted skins.  Skins are files which contains  an
       image and keys definitions.


OPTIONS
       The skinedit program accepts the following options:

       -h, --help
              Display a list of all command line options.

       -v, --version
              Output the version info.


SEE ALSO
       tiemu(1), our TiEmu emulator.
